# LeagueMinimapReminder
Minimap Reminder for League of Legends: customizable sound alerts with variable intervals using hotkeys.
Custom sound alerts: Chose any .mp3 file por the reminder
Variable intervals: adjustu the time between alerts in real-time using hotkeys
Hotkeys: Ctrl + = (increase interval) Crtl + - (decrease interval) F8 (close the program)
Works fullscreen games without showing a console window
Sounds don't overlao, even with short intervals

Open gato.exe .
select a .mp3 sound file using the "Select Sound" button.
Enter the desired interval in seconds
Click "Start Reminder" (the program runs in the background.)
Adjust the interval with Ctrl + = / Crtl + - and close with F8
